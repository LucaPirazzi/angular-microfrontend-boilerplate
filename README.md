## 1. Installazione delle dipendenze

```bash
  npm install
```

**NB**: Se dovessi riscontrare problemi prova ad aggiungere `--legacy-peer-deps`.

```bash
  npm install --legacy-peer-deps
```

### 2. Avvio applicazione

```bash
  npm start
```
